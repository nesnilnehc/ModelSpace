# ModelSpace M2/M3 交付说明

**Date:** 2026-03-05  
**Status:** Implemented

## M2 交付

### 1) 单元聚焦学习路径

- 触发条件：未选中模型且仅选中一个空间单元
- 呈现内容：
  - 单元速览（单元标签、模型数量、推荐起手模型）
  - 可点击学习路径（按推荐模型优先 + 其余模型）
- 点击路径模型名称可直接跳转到该模型详情

### 2) 相关模型分组

在模型详情中新增“相关模型”分组，包含：

- 同单元
- 同类别
- 最近邻

每个分组都支持按钮跳转到对应模型详情。

## M3 交付

### 1) URL 状态恢复与同步

- 支持通过 URL 参数恢复状态：
  - 语言、视角、筛选关键字、模型/单元筛选、图层开关、面板显隐、活动标签页
- 状态变化后会自动 `replaceState` 回写到 URL，便于分享与复现

### 2) 嵌入模式入口

- 新增 `embed.html`，统一转向 `cognitive-model-3d.html?simple=1&embed=1`
- 嵌入模式下隐藏部分非核心控件，保留阅读与视角浏览能力

### 3) 导出命名增强

- 默认：`modelspace-YYYY-MM-DD.png`
- 单元聚焦导出：`modelspace-cell-<cell>-YYYY-MM-DD.png`
- 单类别筛选导出：`modelspace-category-<category>-YYYY-MM-DD.png`

## 验证
