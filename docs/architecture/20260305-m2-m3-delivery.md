# Cognitive Atlas M2/M3 交付说明

**Date:** 2026-03-05  
**Status:** Implemented

## M2 交付

### 1) 单元聚焦学习路径

- 触发条件：未选中对象且仅选中一个空间单元
- 呈现内容：
  - 单元速览（单元标签、对象数量、推荐起手对象）
  - 可点击学习路径（按推荐对象优先 + 其余对象）
- 点击路径对象名称可直接跳转到该对象详情

### 2) 相关对象分组

在模型详情中新增“相关对象”分组，包含：

- 同单元
- 同类别
- 最近邻

每个分组都支持按钮跳转到对应对象详情。

## M3 交付

### 1) URL 状态恢复与同步

- 支持通过 URL 参数恢复状态：
  - 语言、视角、筛选关键字、对象/单元筛选、图层开关、面板显隐、活动标签页
- 状态变化后会自动 `replaceState` 回写到 URL，便于分享与复现

### 2) 嵌入模式入口

- 新增 `embed.html`，统一转向 `cognitive-atlas.html?simple=1&embed=1`
- 嵌入模式下隐藏部分非核心控件，保留阅读与视角浏览能力

### 3) 导出命名增强

- 默认：`cognitive-atlas-YYYY-MM-DD.png`
- 单元聚焦导出：`cognitive-atlas-cell-<cell>-YYYY-MM-DD.png`
- 单类别筛选导出：`cognitive-atlas-category-<category>-YYYY-MM-DD.png`

## 验证
