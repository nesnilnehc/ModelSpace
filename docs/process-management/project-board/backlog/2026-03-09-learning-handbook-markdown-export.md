# 学习手册 Markdown 导出

## Problem / Need

用户希望在 Notion、Obsidian 或本地 Markdown 中按路径进行体系化学习，但 Atlas 当前仅提供浏览器内 3D 探索，无法导出为可离线使用的学习手册。需要按学习路径导出 Markdown，包含模型列表、定义、练习和复习提醒，便于用户按节奏自学。

## Acceptance Criteria

- [ ] 支持按选定学习路径导出 Markdown 文件
- [ ] 导出内容包含：路径名称、模型列表（含 summary、whenToUse）、每个模型的 practice、复习节奏提醒（1 天/3 天/1 周）
- [ ] 输出格式便于 Notion/Obsidian 等工具使用
- [ ] 可选：包含落地页 URL 便于跳回 Atlas 查看详情

## Notes

- 依赖学习路径与练习设计完成
- 与体系化学习主线衔接
